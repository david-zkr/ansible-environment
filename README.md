## Starting 🚀
### Pre-requirements 📋

_You will need:_
````
A virtualization program (VirtualBox, VMWare, ...).
````
````
An RHEL8 ISO. You can download it from Red Hat official website.
````
````
At least 8GB RAM in your computer.
````
